Fourth digit (0000-8000)
========================

====== =========

------ ---------

------ ---------

------ ---------

------ ---------

------ ---------

------ ---------

..  note::

    SunVox does not use values higher than 8000 hex.

    This allows for decimal values 0 to 32768, or -16384 to 16384 with an exact midpoint of 0.
